# InternetSOSplay

Aplicación de escritorio para Windows que mantiene activa tu sesión en redes WiFi con portal cautivo.

---

## Requisitos

- Windows 8.1 / 10 / 11 (64 bits)
- PowerShell 5.1 o superior
- Conexión WiFi con portal cautivo (ClubTotalplay / ClubWiFi)

## Instalación

1. Descarga el instalador desde la sección [Releases](../../releases)
2. Ejecuta `InternetSOSplay-Setup-X.X.X.exe`
3. Si Windows muestra un aviso de seguridad: clic en **"Más información"** → **"Ejecutar de todas formas"**
4. Acepta los términos de uso al primer inicio
